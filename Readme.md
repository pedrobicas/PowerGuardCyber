# FalhasEnergiaCiber

Este projeto em C# simula um sistema de monitoramento de falhas de energia e seus impactos cibernéticos.

## Funcionalidades

- Registro de falhas de energia
- Geração de alertas de segurança
- Log de eventos
- Relatórios de status
- Simulação de ataques cibernéticos

## Como executar

1. Clone o repositório ou baixe os arquivos.
2. Abra o projeto no Visual Studio.
3. Compile e execute o projeto (`F5`).
